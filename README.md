## Overview

Planning an academic semester involves balancing:

- Lecturer workload
- Student enrolment
- Credit hours
- Course offerings
- Programme requirements
- Teaching resources

The AIU Semester Simulation Dashboard provides an interactive platform to model these variables before the semester begins, enabling data-driven academic planning.

## Installation

```bash
git clone https://github.com/zulkiflizaki/aiu-semester-simulation.git
cd aiu-semester-simulation
python -m http.server
```

Open your browser and visit:

```text
http://localhost:8000
```
